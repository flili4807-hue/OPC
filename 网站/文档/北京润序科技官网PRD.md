## 1. 项目背景

北京润序科技有限责任公司已成立，当前需要建设公司官网，作为对外展示公司定位、专业能力、代表案例和合作入口的统一品牌阵地。

公司当前最强的可信资产来自创始人范丽丽的技术与管理复合背景：18 年技术 + 项目管理经验，PMP 认证，CMMI3 评估师，长期负责大型复杂项目交付、战略 PMO、产研流程体系搭建、AI Agent 平台和大模型相关项目落地。

因此，本期官网不宜做成泛泛的企业介绍站，而应定位为“专家背书驱动的 B2B 服务型官网”，重点传达：

1. 北京润序科技能帮助技术型组织建立可复制、可度量、可交付的项目管理体系。
2. 公司具备 AI Agent、AI 大模型平台、音视频、云游戏、复杂项目交付等真实项目经验。
3. 公司可以面向创业公司、AI 团队、技术组织、ToB / ToC 产品团队、服务型 PMO 组织提供 PMO、TPM、Agent 平台项目管理和产研效能咨询服务。

## 2. 一句话定位

### 9.9 项目成功率自评问卷页

#### 目标

让潜在客户通过 24 个问题快速自评项目或团队当前交付健康度，并获得项目成功率评分、CMMI 成熟度映射和治理建议。

#### 页面入口

首页诊断模块的“开始自评”链接进入问卷页。

#### 题目结构

问卷共 8 个维度，每个维度 3 题，共 24 题：

1. 目标与范围
2. 组织与职责
3. 需求与变更
4. 计划与风险
5. 质量与测试
6. 数据与效能
7. 工具链与流程
8. 复盘与持续改进

#### 评分规则

| 项目 | 规则 |
|---|---|
| 单题分值 | 1-5 分 |
| 默认题目分值 | 0 分 |
| 满分 | 120 分 |
| 总分公式 | 已选题分值总和 / 120 × 100，四舍五入取整数 |
| 初始状态 | 所有题目默认 0 分，初始得分为 0 分 |
| 结果展示 | 页面打开即展示默认得分；用户调整任一题后实时更新完整成熟度映射 |
| 数据保存 | V1 不保存用户填写结果，仅在本地页面即时计算 |

#### 结果区间

| 分数 | 判断 | CMMI 映射 | 建议 |
|---:|---|---|---|
| 85-100 | 高成功概率 | L4-L5 | 强化数据驱动和最佳实践沉淀 |
| 70-84 | 可控推进 | L3-L4 | 加强风险前置、发布门禁和复盘闭环 |
| 55-69 | 需要治理 | L2-L3 | 优先补齐计划、依赖、风险和质量机制 |
| 40-54 | 高风险 | L1-L2 | 建议先做项目治理重构 |
| 0-39 | 极高风险 | L1 | 先梳理目标、范围、组织和质量体系 |

#### 验收标准

| 前置条件 | 操作 | 预期结果 |
|---|---|---|
| 用户首次打开问卷页 | 无操作 | 所有题目默认 0 分，显示初始得分 0 分和极高风险 · CMMI L1 |
| 用户选择任意题目分值 | 选择 1-5 分 | 得分、完成题数、进度条实时更新 |
| 用户完成 24 题 | 填完所有题 | 显示对应分数区间、CMMI 映射和治理建议 |
| 用户调整任意题目 | 修改 1-5 分 | 得分、等级、进度条和治理建议实时更新 |
| 用户点击顶部 Logo | 点击 Logo | 返回首页 |
| 移动端 375px 宽度 | 打开问卷页并填写 | 无横向滚动，选择器可点击，评分面板可正常查看 |
